Using this tool is simple.
Just double click the exe file and it will limit every trinket of every mod you have downloaded to 1.

You can edit the config.json to change what it limits trinkets to. If you want every trinket to be unlimited, set "limit" to 0.
You can also add a list of trinkets you DONT want to be limited. By default the money trinkets from the T+ mod are excluded due to potential bugs when limiting them to 1.
You don't need to have a mod installed to place its trinket IDs in the exclude list. The list is just a reference of trinkets to skip if the tool finds them.

The tool also creates a convenient backup of every altered mod that you can easily copy and paste back into the main workshop folder to undo the change.

DO NOT LAUNCH THE GAME WHILE THE TOOL IS RUNNING!!

The terminal window will tell you when it's done. If you launch the game while it's running you run the risk of corrupting the files.

In the event you do something and it breaks, you can redownload your workshop files by closing Steam completely and relaunching it.
Alternatively you can validate the game files by right-clicking the game in your library, selecting Properties > Installed Files > Verify integrity of game files
Either option will cause Steam to automatically re-download the latest version of every workshop mod you have subscribed to.
